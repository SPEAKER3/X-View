# X-View 👁️

![.NET](https://img.shields.io/badge/.NET-8.0-purple.svg)
![Docker](https://img.shields.io/badge/Docker-Supported-blue.svg)

**X-View** — це інтелектуальний веб-сервіс (REST API), призначений для автоматизації витягування структурованих даних із неструктурованих текстів та документів. Використовуючи великі мовні моделі (LLM) та інструкції природною мовою, він усуває необхідність ручного введення даних та написання складних регулярних виразів для парсингу.

## ✨ Основні можливості

- **Розумна екстракція:** Витягуйте сутності, таблиці та конкретні дані з файлів `.txt` та `.pdf` за допомогою простих інструкцій природною мовою.
- **Гнучкий експорт:** Миттєво генеруйте та завантажуйте оброблені дані у форматах `.csv` або `.json` для зручної інтеграції у внутрішні системи або бази даних.
- **Інтерактивне прев'ю:** Візуально перевіряйте витягнуті дані в інтерактивній таблиці перед завантаженням файлу.
- **Історія запитів:** Безпечне зберігання ваших останніх сесій екстракції (до 50 записів) для швидкого доступу та повторного експорту.
- **Безпека акаунтів:** JWT-автентифікація користувачів із безпечним хешуванням паролів за алгоритмом bcrypt.
